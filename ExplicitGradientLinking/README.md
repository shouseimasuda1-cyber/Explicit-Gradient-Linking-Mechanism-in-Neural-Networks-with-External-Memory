# Explicit Gradient Linking Mechanism in Neural Networks with External Memory

This repository contains the code, data, and figures accompanying the paper:

**“Explicit Gradient Linking Mechanism in Neural Networks with External Memory”**  
by Shousei Masuda (2025)

---

## Overview

This work proposes an *explicit gradient linking mechanism* that reconnects broken gradient flow
in neural networks with external memory.  
The method accelerates early-stage learning and reveals new insights into the dynamics of
long-term credit assignment.
